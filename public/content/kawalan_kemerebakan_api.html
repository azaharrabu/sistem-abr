Kawalan Kemerebakan Api

Kawalan Kemerebakan Api Luaran

Kawalan kemerebakan api luaran adalah satu aspek penting dalam keselamatan kebakaran yang bertujuan untuk menghadkan penyebaran api dari satu bangunan ke bangunan lain yang berdekatan, serta menghalang api daripada merebak ke dalam atau di sepanjang permukaan luar bangunan itu sendiri, terutamanya di kawasan yang tidak dilindungi seperti bukaan (contohnya tingkap) dan permukaan yang mudah terbakar. Ini penting untuk melindungi harta benda, mengelakkan kemusnahan berantai, dan memastikan keselamatan penghuni serta anggota bomba. Terdapat tiga kaedah atau cara utama untuk mengawal kemerebakan api luaran iaitu:

  1. Jarak Antara Bangunan: Ini adalah kaedah paling asas dan berkesan untuk mengurangkan risiko penyebaran api secara radiasi haba atau serpihan api antara bangunan. Dengan memastikan jarak yang mencukupi antara dua struktur, haba dari kebakaran di satu bangunan tidak akan cukup intens untuk menyalakan bangunan di sebelahnya. Jarak yang diperlukan sangat bergantung kepada jumlah bukaan seperti tingkap, pintu, atau mana-mana bahagian dinding yang tidak kalis api sepenuhnya, pada sisi bangunan yang menghadap bangunan lain. Semakin banyak bukaan pada dinding bangunan, semakin besar kemungkinan api dan haba akan memancar keluar dari bangunan tersebut semasa kebakaran. Oleh itu, jarak antara bangunan perlu lebih jauh sekiranya lebih banyak bukaan diadakan untuk mengimbangi risiko ini. Kaedah pengiraan jarak antara bangunan ini adalah tertakluk dan boleh dirujuk secara terperinci di Jadual Keenam Undang-Undang Kecil Bangunan Seragam 1984 (UUKBS 1984). Ini memastikan piawaian yang seragam dan selamat dipatuhi dalam perancangan bandar dan pembangunan hartanah.
  2. Bumbung: Bumbung memainkan peranan kritikal dalam mengawal kemerebakan api luaran. Pembinaan bumbung bangunan hendaklah mematuhi beberapa Undang-Undang Kecil (UUK) yang relevan dalam UUKBS 1984 seperti UUK 208, UUK 210, UUK 211 dan Jadual Kelapan. Bumbung memainkan peranan kritikal dalam mengawal kemerebakan api luaran kerana ia adalah permukaan besar yang terdedah kepada percikan api atau haba radiasi dari kebakaran berdekatan. Bumbung juga boleh menjadi laluan untuk api merebak ke bahagian dalam bangunan atau ke bangunan lain jika tidak direka dan dibina dengan betul. Pembinaan bumbung bangunan hendaklah mematuhi beberapa Undang-Undang Kecil (UUK) yang relevan dalam UUKBS 1984:
    1. UUK 208, UUK 210, UUK 211
      Ini adalah peruntukan spesifik dalam UUKBS 1984 yang menggariskan keperluan bagi bahan bumbung, reka bentuk, dan pemasangan untuk memastikan ketahanan api dan mengurangkan risiko penyebaran api.
    2. Jadual Kelapan Undang-Undang Kecil Bangunan Seragam 1984
      Jadual ini menyediakan klasifikasi bahan bumbung berdasarkan ketahanan api atau keperluan spesifik untuk jenis bangunan tertentu.
  3. Dinding Luar: Dinding luar berfungsi sebagai penghalang fizikal yang pertama terhadap penyebaran api dari sumber luaran. Ketahanan api bagi dinding luar perlu dipatuhi mengikut Jadual Kesembilan UUKBS 1984. Jika terdapat salutan (cladding), keperluan UUK 144 hendaklah dipatuhi. Dinding luar berfungsi sebagai penghalang fizikal yang pertama terhadap penyebaran api dari sumber luaran. Ia juga perlu mengawal kemerebakan api di permukaan luarnya sendiri. Dinding luar hendaklah dibina untuk:
    1. Menghadkan risiko penyalaan dari sumber luar
      Ini bermaksud dinding itu sendiri tidak mudah terbakar apabila terdedah kepada api atau haba dari luar.
    2. Menghadkan kemerebakan api ke atas permukaan
      Jika api berjaya menyalakan permukaan dinding, ia tidak boleh merebak dengan cepat atau meluas di sepanjang permukaan tersebut. Ini mengurangkan risiko api masuk ke dalam bangunan melalui dinding atau merebak ke bahagian lain bangunan.
    Ketahanan api bagi dinding luar perlu dipatuhi mengikut Jadual Kesembilan Undang-Undang Kecil Bangunan Seragam 1984. Jadual ini akan menetapkan tempoh masa minimum dinding luar perlu kekal utuh dan menghalang penyebaran api di bawah keadaan kebakaran standard.

Salutan Dinding Luar (Cladding):

Sekiranya terdapat salutan atau cladding pada dinding luar (yang sering digunakan untuk tujuan estetik atau penebat), keperluan UUK 144 hendaklah dipatuhi.
UUK 144 mungkin merujuk kepada jenis bahan yang dibenarkan untuk cladding, ketahanan api bahan tersebut, atau cara pemasangannya untuk memastikan ia tidak menjadi punca penyebaran api yang cepat dan tidak terkawal di bahagian luar bangunan (fenomena "chimney effect" atau "faÃ§ade fire").
Secara ringkas, kawalan kemerebakan api luaran adalah usaha multi disiplin yang melibatkan perancangan ruang yang bijak (jarak antara bangunan), penggunaan bahan binaan yang sesuai dan kalis api (bumbung dan dinding luar), serta pematuhan ketat kepada piawaian dan undang-undang yang ditetapkan seperti UUKBS 1984. Ini semua bertujuan untuk melindungi komuniti dan infrastruktur daripada bahaya kebakaran yang meluas.

Kawalan Kemerebakan Api Dalaman

Kawalan kemerebakan api dalaman merujuk kepada langkah-langkah dan sistem yang direka bentuk untuk menghadkan dan menguruskan penyebaran api serta asap di dalam sesebuah bangunan itu sendiri. Matlamat utamanya adalah untuk melindungi nyawa penghuni, meminimumkan kerosakan harta benda, dan memastikan kestabilan struktur bangunan semasa kebakaran. Kawalan Kemerebakan Api Dalaman, yang terbahagi kepada dua kategori utama iaitu Sistem Perlindungan Pasif dan Sistem Perlindungan Aktif. Kedua-dua sistem ini bekerja secara sinergi untuk memastikan keselamatan kebakaran di dalam bangunan.

Sistem Perlindungan Pasif

Definasi perlindungan pasif ialah unsur-unsur bagi struktur sesuatu bangunan yang berupaya untuk melaksanakan fungsinya dengan sendiri atau berkehendakkan bantuan bagi membendung atau menyekat asap dan api daripada merebak di dalam bangunan semasa kebakaran.

Komponen-komponen Utama Sistem Perlindungan Pasif:
  • Jalan Keluar Keselamatan (Means of Escape): Ini merujuk kepada semua laluan dan pintu kecemasan yang direka untuk membolehkan penghuni bangunan keluar dengan selamat dan cepat bila ada kebakaran. Reka bentuk laluan ni mesti jelas, tak ada halangan, dan mudah diakses oleh semua orang. Ia adalah laluan hidup kita.
  • Pemetakan (Compartmentation): Pemetakan ni bermaksud membahagikan bangunan kepada bahagian-bahagian atau "kompartmen" yang berasingan. Ini dibuat menggunakan dinding, lantai, dan pintu tahan api. Tujuannya adalah untuk mengurung api dan asap dalam satu kawasan sahaja, melambatkan penyebarannya. Ini memberi kita lebih banyak masa untuk lari keluar atau beri masa kepada bomba untuk bertindak.
  • Kekemasan Dalaman dan Siling (Internal Linings and Ceilings): Bahan yang digunakan untuk dinding, lantai, dan siling (langit-langit) dalaman bangunan mesti ada sifat tahan api tertentu. Ini termasuklah guna bahan yang tak mudah terbakar atau yang lambat menyebarkan api. Ini penting untuk elak api merebak dengan cepat melalui permukaan dalaman bangunan.
  • Pengudaraan dan Pencahayaan (Ventilation and Lighting): Walaupun nampak macam sistem yang aktif, sistem pengudaraan pasif dan cahaya semulajadi pun ada peranan dalam keselamatan kebakaran. Pengudaraan yang bagus boleh bantu keluarkan asap dari bangunan, manakala pencahayaan yang cukup (terutamanya lampu kecemasan) sangat penting untuk kita nampak jalan masa nak lari keluar.
  • Akses Menentang Kebakaran (Fire Fighting Access): Komponen ini merujuk kepada penyediaan laluan atau akses yang cukup untuk pasukan bomba masuk ke dalam bangunan dan memadamkan api. Ia boleh termasuk laluan khas, ruang yang cukup untuk peralatan mereka, dan kawasan yang selamat untuk mereka bekerja.
