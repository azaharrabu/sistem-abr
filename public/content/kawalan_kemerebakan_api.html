Kawalan Kemerebakan Api

Kawalan Kemerebakan Api Luaran

Kawalan kemerebakan api luaran adalah satu aspek penting dalam keselamatan kebakaran yang bertujuan untuk menghadkan penyebaran api dari satu bangunan ke bangunan lain yang berdekatan, serta menghalang api daripada merebak ke dalam atau di sepanjang permukaan luar bangunan itu sendiri. Terdapat tiga kaedah atau cara utama untuk mengawal kemerebakan api luaran iaitu:

  1. Jarak Antara Bangunan: Ini adalah kaedah paling asas dan berkesan untuk mengurangkan risiko penyebaran api secara radiasi haba atau serpihan api antara bangunan. Dengan memastikan jarak yang mencukupi antara dua struktur, haba dari kebakaran di satu bangunan tidak akan cukup intens untuk menyalakan bangunan di sebelahnya. Jarak yang diperlukan sangat bergantung kepada jumlah bukaan seperti tingkap, pintu, atau mana-mana bahagian dinding yang tidak kalis api sepenuhnya, pada sisi bangunan yang menghadap bangunan lain. Semakin banyak bukaan pada dinding bangunan, semakin besar kemungkinan api dan haba akan memancar keluar dari bangunan tersebut semasa kebakaran. Oleh itu, jarak antara bangunan perlu lebih jauh sekiranya lebih banyak bukaan diadakan untuk mengimbangi risiko ini. Kaedah pengiraan jarak antara bangunan ini adalah tertakluk dan boleh dirujuk secara terperinci di Jadual Keenam Undang-Undang Kecil Bangunan Seragam 1984 (UUKBS 1984). Ini memastikan piawaian yang seragam dan selamat dipatuhi dalam perancangan bandar dan pembangunan hartanah.
  2. Bumbung: Bumbung memainkan peranan kritikal dalam mengawal kemerebakan api luaran. Pembinaan bumbung bangunan hendaklah mematuhi beberapa Undang-Undang Kecil (UUK) yang relevan dalam UUKBS 1984 seperti UUK 208, UUK 210, UUK 211 dan Jadual Kelapan.
  3. Dinding Luar: Dinding luar berfungsi sebagai penghalang fizikal yang pertama terhadap penyebaran api dari sumber luaran. Ketahanan api bagi dinding luar perlu dipatuhi mengikut Jadual Kesembilan UUKBS 1984. Jika terdapat salutan (cladding), keperluan UUK 144 hendaklah dipatuhi.

Salutan Dinding Luar (Cladding):

Sekiranya terdapat salutan atau cladding pada dinding luar (yang sering digunakan untuk tujuan estetik atau penebat), keperluan UUK 144 hendaklah dipatuhi.
UUK 144 mungkin merujuk kepada jenis bahan yang dibenarkan untuk cladding, ketahanan api bahan tersebut, atau cara pemasangannya untuk memastikan ia tidak menjadi punca penyebaran api yang cepat dan tidak terkawal di bahagian luar bangunan (fenomena "chimney effect" atau "faÃ§ade fire").
Secara ringkas, kawalan kemerebakan api luaran adalah usaha multi disiplin yang melibatkan perancangan ruang yang bijak (jarak antara bangunan), penggunaan bahan binaan yang sesuai dan kalis api (bumbung dan dinding luar), serta pematuhan ketat kepada piawaian dan undang-undang yang ditetapkan seperti UUKBS 1984. Ini semua bertujuan untuk melindungi komuniti dan infrastruktur daripada bahaya kebakaran yang meluas.

Kawalan Kemerebakan Api Dalaman

Kawalan kemerebakan api dalaman merujuk kepada langkah-langkah dan sistem yang direka bentuk untuk menghadkan dan menguruskan penyebaran api serta asap di dalam sesebuah bangunan itu sendiri. Ia terbahagi kepada Sistem Perlindungan Pasif dan Sistem Perlindungan Aktif.

Sistem Perlindungan Pasif

Definasi perlindungan pasif ialah unsur-unsur bagi struktur sesuatu bangunan yang berupaya untuk melaksanakan fungsinya dengan sendiri atau berkehendakkan bantuan bagi membendung atau menyekat asap dan api daripada merebak di dalam bangunan semasa kebakaran.

Komponen-komponen Utama Sistem Perlindungan Pasif:
  • Jalan Keluar Keselamatan (Means of Escape): Merujuk kepada semua laluan dan pintu kecemasan yang direka untuk membolehkan penghuni bangunan keluar dengan selamat dan cepat.
  • Pemetakan (Compartmentation): Membahagikan bangunan kepada bahagian-bahagian atau "kompartmen" yang berasingan menggunakan dinding, lantai, dan pintu tahan api.
  • Kekemasan Dalaman dan Siling (Internal Linings and Ceilings): Bahan yang digunakan untuk dinding, lantai, dan siling dalaman bangunan mesti ada sifat tahan api tertentu.
  • Pengudaraan dan Pencahayaan (Ventilation and Lighting): Sistem pengudaraan pasif dan cahaya semulajadi pun ada peranan dalam keselamatan kebakaran.
  • Akses Menentang Kebakaran (Fire Fighting Access): Merujuk kepada penyediaan laluan atau akses yang cukup untuk pasukan bomba masuk ke dalam bangunan.
